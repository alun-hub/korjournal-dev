Integritetspolicy

Körjournal  ·  Senast uppdaterad: 2026-03-02

Körjournal är en personlig GPS-körjournal som körs lokalt i din webbläsare eller som installerad app på din enhet. Appen samlar inte in, lagrar inte och delar inte personuppgifter med någon extern part, med undantag för det som beskrivs nedan.

Data som behandlas

  • GPS-position — används för att spåra körningar i realtid. Positionsdata lagras enbart lokalt på din enhet (via webbläsarens localStorage) och skickas aldrig till någon server som vi driver.
  • Adresser — när en körning avslutas skickas start- och stoppkoordinaterna till OpenStreetMap Nominatim (nominatim.openstreetmap.org) för omvänd geokodning. Inga personuppgifter utöver GPS-koordinaterna ingår i dessa anrop. Se Nominatims policy.
  • Kartbilder — laddas från OpenStreetMap tile-servrar (tile.openstreetmap.org). Se OpenStreetMaps integritetspolicy.

Lokal lagring

All resedata (datum, tid, sträcka, trängselskatt, anteckningar) sparas uteslutande lokalt på din enhet med webbläsarens localStorage. Inga data synkroniseras till molnet och ingen server lagrar din resehistorik.

Delning av data

Appen delar ingen data med tredje part förutom de tekniska anrop till OpenStreetMap som beskrivs ovan. Det finns inga annonsnätverk, analysverktyg eller spårningscookies.

Behörigheter

  • Plats (GPS) — krävs för att registrera körningar. Platsåtkomst aktiveras bara när du trycker på "Starta körning" och stängs av när körningen avslutas.
  • Skärmlås (Wake Lock) — används för att hålla skärmen aktiv under aktiv körning så att GPS-spårningen inte avbryts.

Dataradering

Du kan radera all lagrad data direkt i appen (välj resor → radera) eller via din webbläsares inställningar under "Rensa webbplatsdata" för körjournal.unixkonsult.se.

Ändringar i policyn

Om policyn uppdateras anges det med ett nytt datum högst upp på denna sida.

Körjournal är en öppen källkodsprojekt utan kommersiellt syfte. Källkod finns på GitHub.
